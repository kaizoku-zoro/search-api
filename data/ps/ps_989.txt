Given integers AA, BB and NN, you should calculate the GCD of AN+BNAN+BN and |A−B||A−B|. (Assume that GCD(0,a)=aGCD(0,a)=a for any positive integer aa). Since this number could be very large, compute it modulo 10000000071000000007 (109+7109+7).
Input

The first line of the input contains a single integer TT denoting the number of test cases. The description of TT test cases follows.
The first and only line of each test case contains three space-separated integers AA, BB and NN.

Output
For each test case, print a single line containing one integer — the required GCD modulo 109+7109+7.
Constraints

1≤T≤101≤T≤10
1≤A,B,N≤10121≤A,B,N≤1012
B≤AB≤A

Subtasks
Subtask #1 (10 points): 1≤A,B,N≤101≤A,B,N≤10
Subtask #2 (40 points): 1≤A,B,N≤1091≤A,B,N≤109
Subtask #3 (50 points): original constraints

                        Sample Input 1
                        





2
10 1 1
9 1 5



                        Sample Output 1
                        





1
2


Explanation
Example case 1: GCD(101+11,10−1)=GCD(11,9)=1GCD(101+11,10−1)=GCD(11,9)=1
Example case 2: GCD(95+15,9−1)=GCD(59050,8)=2GCD(95+15,9−1)=GCD(59050,8)=2
