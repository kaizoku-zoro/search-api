Danya gave integers aa, bb and nn to Archi. Archi wants to compare anan and bnbn. Help Archi with this task.
Input

The first line of the input contains a single integer TT denoting the number of test cases. The description of TT test cases follows.
The first and only line of each test case contains three space-separated integers aa, bb and nn.

Output
For each test case, print a single line containing one integer: 11 if an>bnan>bn, 22 if an<bnan<bn or 00 if an=bnan=bn.
Constraints

1≤T≤1,0001≤T≤1,000
|a|,|b|≤109|a|,|b|≤109
1≤n≤1091≤n≤109


                        Sample Input 1
                        





2
3 4 5
-3 2 4



                        Sample Output 1
                        





2
1


Explanation
Example case 1: an=243an=243 and bn=1024bn=1024.
Example case 2: an=81an=81 and bn=16bn=16.
