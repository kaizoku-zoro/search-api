Almir had a small sequence A1,A2,…,ANA1,A2,…,AN. He decided to make KK copies of this sequence and concatenate them, forming a sequence X1,X2,…,XNKX1,X2,…,XNK; for each valid ii and jj (0≤j<K0≤j<K), Xj⋅N+i=AiXj⋅N+i=Ai.
For example, if A=(1,2,3)A=(1,2,3) and K=4K=4, the final sequence is X=(1,2,3,1,2,3,1,2,3,1,2,3)X=(1,2,3,1,2,3,1,2,3,1,2,3).
A pair (i,j)(i,j), where 1≤i<j≤N1≤i<j≤N, is an inversion if Xi>XjXi>Xj. Find the number of inversions in the final sequence XX.
Input

The first line of the input contains a single integer TT denoting the number of test cases. The description of TT test cases follows.
The first line of each test case contains two space-separated integers NN and KK.
The second line contains NN space-separated integers A1,A2,…,ANA1,A2,…,AN.

Output
For each test case, print a single line containing one integer ― the number of inversions in the sequence XX.
Constraints

1≤T≤1,0001≤T≤1,000
1≤N≤1001≤N≤100
1≤K≤1061≤K≤106
1≤Ai≤1091≤Ai≤109 for each valid ii
